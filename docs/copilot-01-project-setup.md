## 4. Database Setup

### Create PostgreSQL database and users:

```sql
-- Run as PostgreSQL superuser
CREATE DATABASE havas_media;

-- App user (full access)
CREATE USER havas_app WITH PASSWORD '<secure-password>';
GRANT ALL PRIVILEGES ON DATABASE havas_media TO havas_app;
ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT ALL ON TABLES TO havas_app;
ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT ALL ON SEQUENCES TO havas_app;

-- Reader user (Python read-only access)
CREATE USER havas_reader WITH PASSWORD '<secure-password>';
GRANT CONNECT ON DATABASE havas_media TO havas_reader;
GRANT USAGE ON SCHEMA public TO havas_reader;
ALTER DEFAULT PRIVILEGES IN SCHEMA public GRANT SELECT ON TABLES TO havas_reader;
```

## 7. Internal API Middleware

### `app/Http/Middleware/InternalApiAuth.php`
```php
<?php

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class InternalApiAuth
{
    public function handle(Request $request, Closure $next)
    {
        $token = $request->header('X-Internal-Token');

        if (!$token || !hash_equals(config('services.internal_api_token'), $token)) {
            return response()->json(['error' => 'Unauthorized'], 401);
        }

        return $next($request);
    }
}
```

### `config/services.php` — Add:
```php
'internal_api_token' => env('INTERNAL_API_TOKEN'),
```

### Register middleware in `bootstrap/app.php` (Laravel 11):
```php
->withMiddleware(function (Middleware $middleware) {
    $middleware->alias([
        'internal.auth' => \App\Http\Middleware\InternalApiAuth::class,
        'role' => \App\Http\Middleware\CheckRole::class,
    ]);
})
```

## 8. Role Middleware

### `app/Http/Middleware/CheckRole.php`
```php
<?php

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;

class CheckRole
{
    public function handle(Request $request, Closure $next, string ...$roles)
    {
        $user = $request->user();

        if (!$user || !in_array($user->role, $roles)) {
            abort(403, 'Insufficient permissions');
        }

        return $next($request);
    }
}
```

**Usage in routes:**
```php
Route::middleware(['auth:sanctum', 'role:admin'])->group(function () {
    // Admin-only routes
});

Route::middleware(['auth:sanctum', 'role:admin,manager'])->group(function () {
    // Admin + Manager routes
});
```
